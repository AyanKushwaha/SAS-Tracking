Crew Portal Bid User Guide : Leave orientation : Leave window : Leave calendar
Leave calendar
The calendar, located in the top of the Leave window, displays days, weeks and months and additional information such as week numbers and weekends. By default three months are displayed, starting with the start date of the Bid period.
Colour coding
The colour coding used in the Leave calendar is client-specific and indicates for example work patterns or pre-assignments. An infotip with details is displayed by hovering over a colour block in the calendar.
Bid window
Information about the current bid window is displayed on top of the Leave calendar.
The bid window indicates the timeframe during which you can create and submit leave bids for the Bid period displayed in the Leave calendar. The following applies:
•
When the bid window is open, you can create and modify leave bids.
•
When the bid window is closed and the next bid window is defined but not open, a notification is displayed that the bid window is closed.
•
When the bid window is closed and no next bid window is defined, a warning is displayed. No content from any period is shown.
Bid period
Information about the current bid window is displayed on top of the Leave calendar.
The bid period is the period being planned. For example, for the bid period November 1-30, you can submit leave bids during the bid window September 1 to October 31.
Navigation
To navigate in the calendar click the backward arrow or forward arrow .
