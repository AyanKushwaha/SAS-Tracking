Creating career bids
Note
You can create bids until the end date of the bid window.
To create a new career bid in the Career bid work area:
1.
In the Career bid window, select Awarding type tab, for example Contract or Transition.
2.
Select New and from the shortcut menu select bid type.
Bid properties are displayed in a form configured for the selected career bid type, see New Career Bid form for an example.
3.
In the form, specify bid properties by entering the appropriate values for the components and click OK.
The bid is saved and added as the last bid in the bid group for the selected bid type.
