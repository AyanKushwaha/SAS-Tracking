Validation view
The Validation view is located at the bottom of the main window. It displays validation errors and warnings, including a description and which bid that caused the error or warning.
To display errors or warnings:
•
When validations errors or warnings occur, click the validation error icon or validation warning icon to display the contents of the Validation view.
•
To hide the contents of the Validation view, click the icon again.
 
•
If no validation errors or warnings exist, the Validation view shows a confirmation message, for example .
