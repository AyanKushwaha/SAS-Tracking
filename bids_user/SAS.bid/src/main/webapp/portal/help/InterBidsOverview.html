Getting started with Crew Portal Bid
Crew Portal Bid contains a number of tabs. Depending on your configuration, you can access one or more of the following tabs:
•
Activity & Time Off tab, see Activity & Time Off orientation for an overview of the user interface. In this tab you can view, create and modify different types of bids and requests, work with bidding scenarios, get decision support, browse and mark trips, and work with reports. For a detailed description of the procedures, see Activity & Time Off procedures.
•
Leave tab, see Leave orientation for an overview of the user interface. In this tab you can view, create and modify leave bids, see Leave procedures for a detailed description.
•
Career tab, see Career orientation for an overview of the user interface. In this tab you can view, create and modify career bids, see Career procedures for a detailed description.
•
Home tab, used to View personal information.
•
Messages tab, used to View messages.
For a description of general working procedures in the Crew Portal Bid application, as for example log on and log out, printing and sorting and grouping tables, see Working in the user interface.
