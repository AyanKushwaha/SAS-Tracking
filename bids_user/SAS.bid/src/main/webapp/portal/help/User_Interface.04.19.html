Report work area
The Report work area contains reports. Name, modification date and file size are displayed in the Report work area.
You can sort and group the reports in different ways, for example configure which column is displayed, reorder columns, increase or decrease column size, and sort view by column. For more information, see Sorting and grouping tables. Personal settings are used.
Report work area
The Report work area menu has the following command:
Open
Opens the selected report.
