Crew Portal Bid User Guide : Activity & Time Off procedures : Trip marking : Marking/Unmarking trips
Marking/Unmarking trips
You can mark a subset of trips from a larger set in the Trip work area, because you are interested in creating bids for them or to flag the trips you are either interested in as part of a larger search or want to avoid, to make it easier to spot them later on.
Marking trips
To mark trips:
•
In the Trip work area, select one or more trips and select the Mark command, and select a marking property on the shortcut menu: Low, Medium or High.
The trip is marked with the corresponding colour in the Gantt chart, or with a marker in the Mark column in the Table view.
If a trip was already marked, you can apply a different marking colour and the existing mark is overwritten. This setting is saved for the remainder of this session.
You can use marked trips to create a bid, see Creating bids based on marked trips.
Unmarking trips
To unmark trips:
•
In the Trip work area, select one or more trips and select the Unmark command.
The markings from the selected trips are removed. The list of trips displayed is refreshed to only show those that are currently marked.
