Request manager
To open the Request manager, select the Request tab in the main window. The Request manager consists of the Request explorer, the Request work area, and the Calendar. In the Request manager you can enter requests, for example for days off or trips. When you submit the request, it will be awarded or rejected on a first come, first served basis (FCFS).
