Creating requests
To create a request:
1.
In the Request explorer, select a request category. The calendar specific for the request category is displayed and in the Available Days Off overview you can see available slots per day.
2.
You can create requests in the following ways:
•
In the Request work area menu, select New.
A list of the possible request types is displayed, for example Days off or Trips. Select the request type for which you want to create a request.
•
In the Request work area, if available, click one of the request type icons displayed on the toolbar.
•
In the calendar, by selecting a specific date, date range or a weekday.
The Request form is displayed for the selected request type.
3.
Enter data and click OK.
A confirmation window is displayed. In this step you can cancel the request by selecting Cancel.
4.
To confirm the request, click OK.
The request is created and a confirmation of the awarded request is displayed in the Request work area. If the request was rejected, the state of the request is set to rejected and a reason is displayed.
Note
When more users enter the same request at the same time, a conflict can occur. The system will try to save your request a number of times, but you will get an error message if the system fails to save your request. In that case, it is recommended to try again after a few minutes.
