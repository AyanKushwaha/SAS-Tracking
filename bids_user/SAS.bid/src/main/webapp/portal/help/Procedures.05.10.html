Creating standard bids
You can create standard bids, such as Buddy bids, Layover bids, using the Standard Bid dialogue in the Bid manager or for specific dates, weekdays or date ranges in the Calendar. The Standard Bid dialogue is used for basic bid types with few properties.
Creating bids in the Bid manager
To create a new bid in the Bid manager:
1.
In the Bid explorer, select a bid scenario in which you want to create a new bid.
2.
You can create bids in the Bid manager in two ways:
•
In the Bid explorer, select the Bid menu.
A list of the possible bid types or options is displayed, see Bid types. Select the bid type for the bid you want to create.
•
In the Bid work area, if available, click one of the bid type icons displayed on the toolbar.
Bid properties are displayed in a Dynamic Bid dialogue configured for the selected bid type, see New Trip Bid form for an example.
3.
In the form, specify bid properties by entering the appropriate values for the components and click Save.
The bid is saved and added to the bid scenario as the last bid in the list.
Creating calendar-based bids
To create calendar-based bids:
1.
In the Calendar, select one of the following options within the current bidding period in the calendar:
•
a specific date
•
date range
•
a weekday.
A list of the possible bid types is displayed.
2.
Select the bid type for the bid you want to create.
Bid properties for the selected bid are displayed in a Standard Bid Dialogue specific for the bid type, see New Bid [type] form for an example.
3.
In the Bid Type form, specify bid properties by entering the appropriate values for the components and click Save.
The bid is saved in the bid scenario.
