Bid explorer
In the Bid explorer, the Current, Standing, Template, Global and History bid categories are displayed, see Bid categories for details. Bid categories contain bid scenarios, see Bid scenarios. When you open the Bid manager, the current bid category is displayed in the Bid manager, along with the first bid scenario and its contents displayed in the Bid work area.
•
Open a bid category by clicking it. Only one bid category can be open at the same time.
•
Open a bid scenario by clicking it. Bids in the bid scenario are displayed in the Bid work area. Only one bid scenario can be open at a given time.
Menu bar
The Bid explorer contains the Bid Scenario menu, Edit menu and Bid Tools menu.
Note
Not all commands in the menus are available for each bid category.
Bid Scenario menu
The Bid Scenario menu has the following commands:
New
Creates a new bid scenario, see Creating bid scenarios.
View details
Shows details on All bid scenarios or Selected bid scenario, see or create Finalized Bid Scenarios, see Viewing bid scenario details.
Edit menu
The Edit menu has the following commands:
Rename
Renames the selected bid scenario, see Modifying bid scenarios.
Copy
Copies the selected bid or bid scenario to the session clipboard. To paste use Paste before or Paste after. See Copying bids.
Paste before
Pastes the copied bid scenario before the selected row, see Copying bids.
Paste after
Pastes the copied bid scenario after the selected row, Copying bids.
Paste
Available when Paste before or Paste after commands are not available.
When using Paste in the Bid scenario list, the position of the new bid scenario will be last.
Delete
Deletes a bid scenario, see Deleting bid scenarios.
Finalize
Marks the selected bid scenario as ready, see Setting the status.
Set to Draft
Marks the selected bid scenario as draft. Only available when the bid scenario is finalized, see Setting the status.
Bid Tools menu
The Bid Tools menu has the following commands:
Optimization-on-Demand
Sends bids in a bid scenario to the optimizer, see Optimization-on-demand. After the generation, the optimization report is available in Report viewer.
Show Bid Impact Overview
Shows Bid Impact overview in the Calendar, see Showing/Hiding Bid Impact overview.
Hide Bid Impact Overview
Hides Bid Impact overview in the Calendar, see Showing/Hiding Bid Impact overview. This command is only available after you have selected the Show Bid Impact Overview command.
View Trip Points
Shows points that the optimizer will give trips that match your bids in the selected bid scenario, see Viewing Trip Points.
Calculate Bid Impact
Gets statistics for the selected bid scenario, see Viewing bid impact.
