Creating Specific Pattern bids
To create a trip bid based on a specific pattern:
1.
In the Bid explorer, select a bid scenario in which you want to create a new bid.
2.
Select a bid type that contains Show matching trips/blocks, for example New Trip bid. The Trip Bid window is displayed.
3.
In the Trip Bid window, click Show matching trips/blocks.
The Trip window appears. Alternatively, display the Trip window by Viewing Trip Points or Viewing matching trips.
4.
In the list, select the trip that you wish to bid for and in the toolbar menu select [...] Bid. The Bid command displayed is dependent on the bid type.
A window appears, displaying trip properties for the selected trip.
5.
Click OK to create your Specific Pattern bid.
The bid is added to the bid scenario as the last bid in the list.
6.
When you are finished creating Specific Pattern bids, close the Matching Trips/blocks window.
Note
You can copy, paste and delete a Specific Pattern bid, but not modify it.
You can only create one Specific Pattern bid at a time.
Specific Pattern bids are not valid until further notice (UFN).
