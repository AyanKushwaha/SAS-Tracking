Bid work area
The Bid work area contains an overview of bids and bid details, including start date and type of bid (work, off duty).
Viewing bid details
Click the (+)-sign to display details on each bid. To hide details, click the minus (-) sign.
An infotip on bid type (work, off-duty) is displayed by hovering over the bid type overview.
Sorting and grouping bids
You can sort and group the bids in different ways, for example configure which column is displayed, reorder bid columns, increase or decrease bid column size, and sort view by column. For more information, see Sorting and grouping tables.
Re-ordering bids
You can re-order bids in different ways:
•
Select a bid and move it to a different position relative to other bids by dragging it to the new position.
•
Select a bid and select Edit> Move up or Edit> Move down to move the bid up or down.
Menus
The Bid work area contains the Bid menu and the Edit menu.
Bid menu
The Bid menu has the following commands:
New...
Creates a new bid of a certain bid type, for example layover. When you select a bid type, a form for this bid type is displayed, see Bidding. See Bid types for an overview of available bid types.
Open
Opens the bid that you selected in the Bid work area. The form for this bid opens and you can now modify the bid, see Modifying bids.
Edit menu
The Edit menu has the following commands:
Open
Opens the Global Bids form, see Modifying Global bids. This command is only available when a Global bid is selected.
Copy
Copies the selected bid to the session clipboard, see Copying bids.
Paste before
Pastes the copied bid before the selected row, see Copying bids.
Paste after
Pastes the copied bid after the selected row, Copying bids.
Paste
Available when Paste before or Paste after commands are not available.
When using Paste in the Bid list, the position of the new bid will be according to the current sorting.
Move up
Moves the selected bid up, see Re-ordering bids.
Move down
Moves the selected bid down, see Re-ordering bids.
Delete
Deletes the selected bid, see Deleting bids.
Toolbar
Dependent on the installation, a number of icons for different bid types are available on the toolbar. To create a bid for a specific bid type, click the icon.
