Crew Portal Bid User Guide : Working in the user interface : View messages
View messages
When you log in you can view and acknowledge messages.
To view messages:
•
In the crew portal window, select the Messages tab.
Messages, if any, are displayed in the Messages window. Each message is displayed on a row.
To acknowledge messages:
•
In the Messages window, select the Acknowledge messages command.
The messages are acknowledged.
