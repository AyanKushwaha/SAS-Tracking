Creating bid scenarios
To create a bid scenario:
1.
Select a bid category.
2.
In the Bid explorer, select Bid Scenario> New.
The Create Bid Scenario form is displayed.
3.
Enter required values and click OK.
The new bid scenario is created and added to the list of bid scenarios.
