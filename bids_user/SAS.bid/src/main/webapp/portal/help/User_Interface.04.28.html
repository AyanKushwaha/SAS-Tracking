Global Bids form
The Global Bids form is used to modify Global bids, see Modifying Global bids.
Parameters
The Global Bids form contains client-specific parameters.
Buttons
The form has the following buttons:
Save
Saves the Global bid.
Cancel
Closes the form without any action.
