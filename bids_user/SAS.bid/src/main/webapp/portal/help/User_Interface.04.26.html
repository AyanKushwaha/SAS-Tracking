New Filter form
The New Filter form is used to search for trips and to create and modify filters, see Creating filters and Modifying filters.
Parameters
The New Filter form contains client-specific parameters.
Buttons
The New Filter form contains the following buttons:
Search
Submits the search request and displays a list of trips matching the filter criteria in the Trip work area. Results are automatically saved as a filter in the Trip explorer.
Set Name & Search
Allows you to enter name of the filter before submitting the search request. Displays a list of trips matching the filter criteria in the Trip work area. Results are automatically saved as a filter in the Trip explorer.
Cancel
Closes the form without any action.
