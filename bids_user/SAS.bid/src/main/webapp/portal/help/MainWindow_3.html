Crew Portal Bid User Guide : Career orientation : Career window
Career window
When you have logged on to Crew Portal Bid and select the Career tab, the Career window is displayed. The Career window contains the following main items:
•
Awarding type
•
Bid period
•
Bid window
•
Career bid work area
 
Awarding type
The Career bid window contains multiple Awarding type tabs, for example Transition or Contract. Each awarding type is connected to one or multiple crew groups, a bid period and bid window. Each awarding type contains one or more bid types, for example awarding type Transition can contain bid type Transfer to bid for transfer to another base, or bid type Position to bid for another qualification and/or rank. For each bid type you can create one or more bids that are grouped together in a bid group. If enabled you can rearrange bids in priority order within a bid group. The bids in the bid group are then processed in priority order.
Awarding types and corresponding bid types and bid groups are client-specific, please contact your ambassador user.
Bid period
Information about the current bid period for the selected Awarding type is displayed on top of the Career bid work area, directly above the menu bar.
The bid period is the period being planned. For example, for the bid period November 1-30, you can submit bids during the bid window September 1 to October 31. The bid period applies to all bid groups within the selected Awarding type. The bid period time is displayed in the time zone of the crew home base.
Bid window
Information about the current bid window for the selected Awarding type is displayed on top of the Career work area, directly above the menu bar. The bid window indicates the timeframe during which you can create and submit career bids for the Bid period. The bid window applies to all bid groups within the selected Awarding type. The bid window time is displayed in the time zone of the crew home base.
The following applies:
•
When the bid window is open, you can create and modify career bids.
•
When the bid window is closed and the next bid window is defined but not open, a notification is displayed that the bid window is closed.
•
When the bid window is closed and no next bid window is defined, a warning is displayed. No content from any period is shown.
