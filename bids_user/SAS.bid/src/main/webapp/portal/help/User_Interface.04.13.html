Request work area
The Request work area contains detailed information about requests that you have created and submitted: request type, start and end date for the request, status (for example awarded or rejected) and a reason if the request was rejected. Only details for the selected request type are displayed. Dependent on the installation, you can delete information from the request work area (for example rejected requests), contact your ambassador user for details.
Menu
The Request work area menu has the following command:
New
Creates a new request, see Creating requests.
Toolbar
Dependent on the installation, a number of icons for different request types are available on the toolbar. To create a bid for a specific request type, click the icon.
