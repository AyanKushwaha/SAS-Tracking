Viewing bid scenario details
To view and print a record of all bid scenarios, selected bid scenarios or a finalized bid scenarios:
1.
In the Bid explorer, select one or more bid scenarios in a bid category and select Bid Scenario> View Details. Select one of the following options:
•
All bid scenarios when you want to view details on all bid scenarios in print layout
•
Selected bid scenario when you want to view details on the selected bid scenario in print layout.
•
Finalized Bid Scenarios when you want to view a report of all currently finalized bid scenarios and their bids in print layout, see Finalizing bids. You have the following options:
•
To print the report, see Printing.
•
To close the report, click Close.
