Trip Points window
The Trip Points window is used to display trip points and points per day for trips that match bids in the selected bid scenario, see Viewing Trip Points.
The Trip Points window displays the points that the optimizer will give your trips that match bids in the selected bid scenario. The Trip Points column shows how much the optimizer prioritizes a certain trip compared to the other trips for your roster. The Daily value column shows how many points per day the trip is worth.
Tools menu
The Tools menu has the following commands:
Mark
Marks priority of a selected trip, select Low, Medium, High, or Clear the marking, see Marking/Unmarking trips. You must select one or multiple trips before the command is available.
View Details
Shows details on one or more trips. Select All or Selected trips. See Viewing trip details.
Filter
You can filter the results in the Trip Points window. Use the following commands:
Filter
Enter arguments in the Filter text box and select the command. The result is displayed in the Trip Points window.
Restore
Restores the contents in the Trip Points window and clears the Filter text box.
