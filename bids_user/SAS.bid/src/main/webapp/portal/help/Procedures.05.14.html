Modifying bids
To modify bids:
1.
In the Bid work area, select the bid you want to modify and select Bid> Open, alternatively double-click the bid.
Bid properties for the selected bid are displayed in a form specific for the bid type, see New Bid [type] form for an example.
2.
Enter changes and click Save.
The changes are saved and the bid is updated in the Bid work area.
Note
You cannot modify a Specific Pattern bid, see Creating Specific Pattern bids.
