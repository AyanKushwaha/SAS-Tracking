Vacation bid types : Join Vacation
Join Vacation
Join vacation bids can be placed by some crew groups in Autumn season. A valid joinvacation bid must include all VA/F7 days that are available on 01Sep.
Input
Alt 1 (From)
Select your preferred start date of vacation period.
Alt 1 (To)
Select your preferred end date of vacation period.
Alt 2 (From)
Select alternative start date to preferred start date.
Alt 2 (To)
Select alternative end date to preferred start date.
Etc...
As above.
