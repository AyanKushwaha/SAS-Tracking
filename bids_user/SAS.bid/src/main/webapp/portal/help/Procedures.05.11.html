Creating dynamic bids
You can create combine different bid preferences, such as Layover and Report Time bids, into a single bid using the Dynamic Bid dialogue in the Bid manager. The Dynamic Bid dialogue is used for complex bid types where multiple properties can be combined.
Creating bids in the Bid manager
To create a new bid in the Bid manager:
1.
In the Bid explorer, select a bid scenario in which you want to create a new bid.
2.
In the Bid work area, if available, click one of the bid type icons displayed on the toolbar.
Bid properties are displayed in a Dynamic Bid dialogue configured for the selected bid type, see New Trip Bid form for an example.
3.
In the form, specify bid properties by entering the appropriate values for the components and click Save.
The bid is saved and added to the bid scenario as the last bid in the list.
