Explorer
The explorer area displays different views, dependent on which tab you select in the main window. Dependent on which tab you select, Explorer displays:
•
Bid explorer containing bid categories and bid scenarios.
•
Request explorer containing request categories.
•
Trip explorer containing filters.
•
Report explorer containing report categories.
Each explorer has its own menu selection.
Bid window
The bid window indicates the timeframe during which you can create and submit bids for the Bid period displayed in the Calendar. For example, for the bid period November 1-30, you can submit bids during the bid window September 1 to October 31. The bid window is displayed in the Bid explorer. The following applies:
•
When the bid window is open, you can create and modify bids in all bid categories.
•
When the bid window is closed and the next bid window is defined but not open, a notification is displayed that the bid window is closed. Depending on the configuration, you can create and edit bids in the Standing and Template bid category.
•
When the bid window is closed and no next bid window is defined, a warning is displayed. No content from any period is shown.
