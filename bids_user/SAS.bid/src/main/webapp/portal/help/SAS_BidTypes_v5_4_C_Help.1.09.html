PBS Bid types : Flight
Flight Bid
Use this bid to request a specific flight.
Input
Flight Number
Type the requested flight number, i.e. SK903 (ARN-EWR) or choose a flight number with the corresponding origin-destination pair from the drop down menu.
Departure Date
Choose departure date for the requested flight in the drop down menu. The departure dates that are presented reflects the departure dates for the selected flight number given month.

Please note: Also for airports outside Scandinavian time zones, departure dates is always local time at departure airport.
