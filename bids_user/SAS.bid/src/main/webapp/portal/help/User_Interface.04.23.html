New Advanced Trip Bid form
This form is used to create new bids for a period or one or more weekdays, based on filters, marked trips or the Roster Simulator line, see Creating advanced bids.
Parameters
The form contains the following parameters:
Period
Select a start and end date from the calendar shortcut menu.
Weekday
Weekdays for which the bid is valid. Select one or multiple weekday by clicking them.
Priority
Select a value from the shortcut menu.
Trip search
Select a filter from the shortcut menu. Select the Avoid check box when you want to create a bid avoiding the selected filter results.
Marked trips
Select a marker from the shortcut menu. Select the Avoid check box when you want to create a bid avoiding the selected filter results.
Roster Simulator Line
Select Roster Simulator Line to base the trip on the Roster Simulator line, see Roster Simulator.
Buttons
The form has the following buttons:
Show matching trips
Displays list of matching trips in the Trip window, see Trip window.
Save
Saves the bid.
Cancel
Closes the form without any action.
