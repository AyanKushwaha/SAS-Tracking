Modifying filters
To modify filters:
1.
In the Trip explorer, select the filter you want to modify and select Filter> Open.
The Open Filter form is opened, see the New Filter form for a parameter description.
2.
Modify the search criteria and click Search.
A list of trips is displayed in the Trip work area, matching the criteria entered. The search form is automatically saved.
