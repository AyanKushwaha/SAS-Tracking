New Leave Bid form
The form is used to create and modify a leave bid, see Creating leave bids and Modifying leave bids.
Parameters
The form contains the following parameters:
Period
Select a start and end date from the calendar shortcut menu. The default start date is the first day in the bid period. The selection possible can be dependent on work patterns.
Alt 2
Select a start and end date from the calendar shortcut menu. This period is considered the second alternative, if the period in Period cannot be granted.
Alt 3
Select a start and end date from the calendar shortcut menu. This period is considered as the third alternative, when neither the interval selected in Period nor the period selected in Alt 2 can be granted.
Buttons
The form has the following buttons:
OK
Saves the bid.
Cancel
Closes the form without any action.
