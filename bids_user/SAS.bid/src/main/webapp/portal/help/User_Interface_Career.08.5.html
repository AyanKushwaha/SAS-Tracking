New Career Bid form
The form is used to create and modify a career bid, see Creating career bids and Modifying career bids.
Parameters
The form contains one or more of the following parameters dependent on bid type, for example:
•
base (Transfer)
•
qualification and rank (Position)
•
parttime (percentage) (Contract).
Buttons
The form has the following buttons:
OK
Saves the bid.
Cancel
Closes the form without any action.
