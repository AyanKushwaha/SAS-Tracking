Trip work area
The Trip work area displays filters and trip search results in a Table view or a Gantt chart.
 
Show trips in Table view or Gantt chart
You can display trips in different formats:
•
To display trips in a Table view, click the Table view icon .
In the Table view you can control how the information is displayed, for example which column is displayed, reorder columns, increase or decrease column size, and sort view by column. For more information, see Sorting and grouping tables. Personal settings are used. In the Table view you can also show trips on priority by selecting a value from a shortcut menu.
•
To display trips in a Gantt chart, click the Gantt icon .
The Gantt chart shows how trips are distributed over time. Dependent on the criteria selected, trips are displayed chronologically based upon departure date/time. In the Gantt chart you can also show marked trips by selecting a value from a shortcut menu and sort trips on different criteria, for example start date, trip ID and points.
Displaying detail trip information
To display detail trip information, hover over the trip.
Selecting trips
In the Table view, you can select individual or multiple rows:
•
select individual rows by using the Ctrl-key
•
select multiple trips in consecutive rows by using the Shift-key in between selecting the first and last trips.
Tools menu
The Tools menu has the following commands:
Mark
Marks priority of a selected trip, select Low, Medium, High, or Clear the marking, see Marking/Unmarking trips. You must select one or multiple trips before the command is available.
Add Trips
Adds selected trip(s) to the Roster Simulator line. In the Gantt chart, you can also double-click a trip to add it to the Roster Simulator line, see Adding trips to Roster Simulator line.
View Details
Shows details on one or more trips. Select All or Selected trips. See Viewing trip details.
