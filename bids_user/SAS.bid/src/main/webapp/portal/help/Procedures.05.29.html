Setting the status
To mark the bid scenarios ready to be processed for roster generation:
1.
In the Bid explorer, select a bid scenario in a bid category and select Edit> Finalize.
Bid scenarios that are marked as ready will be processed by the rostering system. A confirmation window is displayed.
2.
To confirm, click Yes.
The bid scenario is marked as finalized in the Bid explorer.
Note
After you marked the bid scenario as finalized you cannot make any changes to the bids. To make changes you have to apply the Edit> Set to Draft command.
