Career bid work area
In the Career bid work area you can create, modify, and view bids that you have submitted for the current bid period.
The Career bid work area contains an overview of bids organized per bid type and bid details. When the bid window is closed, your bids are validated in Jeppesen Manpower Planning and the bids in the Career bid work area are cleared. You can print a receipt of the bids before the bid window is closed, see Print receipt.
 
Viewing bid type details
•
To display details for each bid type, click the plus (+)-sign.
•
To hide details, click the minus (-) sign.
Sorting and grouping bids
You can sort and group the bids in different ways, for example configure which column is displayed, reorder bid columns, increase or decrease bid column size. For more information, see Sorting and grouping tables.
Re-ordering bids
The bid order indicates the award priority. If enabled for the bid type, you can re-order bids within the bid group in different ways:
•
Select a bid and move it to a different position relative to other bids by dragging it to the new position. A green line indicates the approved target area that you move the bid to.
•
Select a bid and select Edit> Move up or Edit> Move down to move the bid up or down. The bid last in the list has the lowest priority.
Menus
The Bid work area contains the New bid menu, the Edit bid menu and the Print receipt command.
New bid menu
The New bid menu has the following commands:
New
From the shortcut menu, select a bid type to create a new career bid, see Creating career bids.
Edit bid menu
The Edit bid menu has the following commands:
Open
Opens the bid that you selected in the Career bid work area. The form for this bid opens and you can now modify the bid, see Modifying career bids.
Move up
Moves the selected bid up, see Re-ordering bids.
Move down
Moves the selected bid down, see Re-ordering bids.
Delete
Deletes the selected bid, see Deleting career bids.
Print receipt
This command prints a receipt of the bids in the selected Awarding type tab, see Print receipt.
