Crew Portal Bid User Guide : Working in the user interface : Log on and log out
Log on and log out
To log on to the system:
1.
Go to the dedicated Crew Portal website, contact your ambassador user for details.
2.
If requested, enter user name and password and select Sign In.
3.
On the CMS Role Selection page, select role and select Sign In.
The Crew Portal is displayed.
4.
To work in Crew Portal Bid, select a tab, for example the Activity & Time Off tab. The Activity & Time off main window is displayed, loaded for the current Bid period.

Toggle the User icon in the top-right of the Crew Portal and click Logout.
