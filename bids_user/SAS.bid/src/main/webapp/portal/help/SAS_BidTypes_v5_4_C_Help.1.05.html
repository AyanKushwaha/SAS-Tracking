PBS Bid types : Compensation days
Compensation days
You can bid the following bid types (depending on your crew group)
• F3
• F3S (only pilots)
• F7S (only cabin attendants). Note this is not an instant reply bid.
• F0 (only cabin attendants)
• F31
• F32 (only senior Danish cabin attendants)
• F33
• F35 (only cabin attendants)
Input
Type of Day
Select type of day.
From
Select the date from the calendar or write it in the format ddmmmyy
Period
Select number of days
 
This is a preferential bid that you cannot give points.
Note
F7S Instant reply is a separate bid type. If you want to place an F7S request, then go to Instant reply requests.
