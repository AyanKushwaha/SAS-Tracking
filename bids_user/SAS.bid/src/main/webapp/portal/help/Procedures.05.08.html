Bidding
You can create standard or dynamic bids, such as Buddy bids, Layover bids, in the Bid manager. You can use the Bid Manager to create bids for specific dates, weekdays or date ranges, but it is faster to do this directly in the Calendar. The Standard Bid dialogue is used for standard, basic bid types with few properties, while the Dynamic Bid dialogue is used for complex bid types where multiple properties can be combined.
You can also create bids based on filters, marked trips or Roster Simulation line, see Creating advanced bids.
You can create bids based on a specific pattern of a trip or block, see Creating Specific Pattern bids.
The basic procedures that you can perform for bids are:
