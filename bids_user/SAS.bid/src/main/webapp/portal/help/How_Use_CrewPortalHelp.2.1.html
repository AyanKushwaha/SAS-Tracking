How to use Help
 
Help is the first place you should turn to for assistance when working in a system from Jeppesen. It quickly provides information about the product. This chapter describes how to use Crew Portal Help.
To start a help system in the Crew Portal, do one of the following:
•
In the main window, select the Bids Help command.
The start page for Crew Portal Bid User Guide help is displayed in a browser.
•
Additionally, when you have access to the Administration tab:
In the main window select the Admin help command.
The start page for the Crew Portal Administration User Guide help is displayed in a browser.
•
In a work area, explorer, or form click . Help displays information about the work area, explorer or form. Note: This function is not available in Crew Portal Bid Administrator.
Help is launched in a tab or browser. To find a particular help topic, you can:
• select the Contents tab to browse through topics by category
• select the Index tab to see a list of index entries
• select the Search tab to do a full-text search.
