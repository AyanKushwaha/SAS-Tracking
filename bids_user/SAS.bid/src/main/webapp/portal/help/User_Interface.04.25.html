New Bid [type] form
For each bid type a form is displayed when you create or modify a bid, see Creating standard bids and Modifying bids. This form uses the Standard Bid Dialogue, used for basic bid types with few properties. The form can contain the following parameters and buttons:
Parameters
Dependent on the bid type, the form contains parameters such as Period, Priority, Flight number, Report time, etc.
Buttons
The form has the following buttons:
Show matching trips
Displays list of matching trips.
Save
Saves the bid.
Cancel
Closes the form without any action.
For an overview of bid types and their properties, see Bid types.
