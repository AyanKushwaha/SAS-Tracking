Optimization-on-demand
Use optimization-on-demand when you want to check that a bid scenario is plausible using all the rules in the optimizer, but without taking other crew's bids into consideration. To perform optimization-on-demand:
1.
In the Bid explorer, select the bid scenario and select Tools> Optimization-on-Demand to perform optimization-on-demand.
Queue information including estimated processing time is displayed. The request is placed in a queue for the optimizer. The bid scenario is marked with .
2.
At some point, the Rostering Optimizer retrieves the request from the queue and processes the request.
When the results are ready, the bid scenario is marked with . The optimization result is published as a report, including a list of trips comprising the reference roster and roster information.
3.
In the Report viewer, open the report.
When the bid scenario name is italic, it means that the trips are being optimized. A bold font indicates that it is finished but has not yet been read and a regular font indicates that the report has been read.
4.
You can also print the report, see Printing.
