Modifying leave bids
To modify bids:
1.
In the Leave bid work area, select the bid you want to modify and select Open, alternatively double-click the bid.
Bid properties for the selected bid are displayed in a form specific for the bid type, see New Leave Bid form for an example.
2.
Enter changes and click OK.
The changes are saved and the bid is updated in the Leave Bid work area.
Note
You can modify leave bids until the end date of the Bid window.
