PBS Bid types : Time Off
Time Off
Use this bid to request time off at your home base (local time). You can use this bid either to request freedays as in Example 1 or to request specific check in/check out time on specific days in Example 2. A maximum of 3 consecutive time off days (or 72 hours) can be selected per bid category (low, medium, high), to provide for fairness between crew.
Shorter periods have higher chance of success.
Input
From
Select the date from the calendar or write it in the format ddmmmyy. Specify the time of day from which you want the time off period to start.
To
Select the date from the calendar or write it in the format ddmmmyy. Specify the time of day to which you want the time off period to end.
Priority
Select bid priority â€“ High, Medium or Low

The priority method works as follows:

If possible, High will always be prioritized above Medium and Low. The same applies to a Medium bid over a Low bid. A High bid also has higher priority than a Medium and Low bid combined.
 
Example 1
Request of time off between 07dec17 00:00 - 10dec17 23:59. This bid will be considered granted if you receive freedays 07dec17 - 10dec17.
 
Example 2
Request of time off between 22dec17 16:00 - 26dec17 08:00. This bid will be considered granted if you on December 22 check out 16:00 at the latest, followed by freedays 23-25dec and if you check in later than 08:00 on December 26.
