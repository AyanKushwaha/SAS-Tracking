Request explorer
The Request explorer contains one or more request categories for which you can create and view requests, as for example days off or trips. Each request category has its own request window indicating the timeframe during which you can create and submit requests for the request period displayed in the calendar.
When you open the Request manager, the topmost request category is selected by default.
When you select the Request manager, the Calendar displays the request period for the selected request category. Available slots as for example days off for the selected request category are displayed in the Available Days Off overview.
