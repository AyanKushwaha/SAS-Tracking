Creating leave bids
To create a new leave bid in the Leave bid work area:
1.
In the Leave bid work area, select New bid and from the shortcut menu select bid type.
Bid properties are displayed in a form configured for the selected leave bid type, see New Leave Bid form for an example.
2.
In the form, specify bid properties by entering the appropriate values and click OK.
The leave bid is saved and added as the last bid in the list.
Note
You can create leave bids until the end date of the Bid window.
