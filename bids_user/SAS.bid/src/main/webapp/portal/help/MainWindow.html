Main window Activity & Time off
When you have logged on to Crew Portal Bid, the main window is displayed in the Activity & Time off tab. The main window contains the Bid, Trip, Request and Report tab.
•
The Bid tab provides access to the Bid manager.
•
The Request tab provides access to Request manager.
•
The Trip tab provides access to Trip browser.The Report tab provides access to Report viewer.
Each tab contains the same areas: Calendar, Explorer, Work area, and Validation view.
