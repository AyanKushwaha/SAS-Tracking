New Trip Bid form
The form is used to create a bid based on trip properties. This is an example of a dynamic bid type, where the Dynamic Bid dialogue is used for complex bid types where multiple properties can be combined, see Creating dynamic bids.
Parameters
The form contains the following parameters:
Period
Select a start and end date from the calendar shortcut menu.
Priority
Select a value from the shortcut menu.
Note Only applicable to weighted fair share.
Trip
 
Start station
Start station for the trip. Click the (+)-sign and select a value from the shortcut menu. To remove the Start station as selection criterion, click .
Stop station
Stop station for the trip. Click the (+)-sign and select a value from the shortcut menu. To remove the Stop station as selection criterion, click .
End time
End time interval for the trip. Click the (+)-sign and select a start and end value from the shortcut menus. To remove the End time as selection criterion, click .
Start time
Start time interval for the trip. Click the (+)-sign and select a start and end value from the shortcut menus. To remove the Start time as selection criterion, click .
Time
 
TAFB
Time away from base. Select start and end value from the shortcut menus. To remove TAFB as selection criterion, click .
Work hours
Interval for number of work hours. Select start and end value from the shortcut menus. To remove Work hours as selection criterion, click .
Buttons
The form has the following buttons:
Show matching trips
Displays list of matching trips.
Save
Saves the bid.
Cancel
Closes the form without any action.
