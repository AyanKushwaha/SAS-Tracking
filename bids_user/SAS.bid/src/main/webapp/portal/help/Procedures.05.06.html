Copying bid scenarios
To copy a bid scenario and its bids to another bid category, or within the same bid category:
1.
In a bid category, select a bid scenario and select Edit> Copy.
The bid scenario is copied to the session clipboard.
2.
Navigate to the target bid category in which the bid scenario will be copied and select a bid scenario.
Dates or periods are applied to the bids as applicable for the new bid category. For example template or standing bids will take on dates for the current bid period when copied to the Current bid category.
3.
To paste the copy of the bid scenario, select Edit> Paste before or Edit> Paste after to paste the copied bid scenario before or after the selected bid scenario. If these commands are not available, select Edit> Paste.
The copy of the bid scenario including its bids, is inserted and the list of bid scenarios is updated. The copy is called CopyofORIGINALNAME.
4.
Repeat this for each bid category to which the bid scenario is to be copied.
