Viewing trip details
To view and print a record of trip details:
1.
In the Trip work area, select one or more trips and select Tools> View Details. Select one of the following options:
•
All or
•
Selected.
A Trip report is displayed for the selection. You have the following options:
•
To print the report, see Printing.
•
To close the report, click Close.
