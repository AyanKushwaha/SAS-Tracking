Printing
You can print the following information:
•
Reports
•
Crew member information (personal page)
•
Bid work area: all bids within a bid scenario, all bids within a bid category for their respective bid scenarios, and trips matching a bid or bid statistic.
•
Trip work area: trips displayed in search results (for new or a saved search), trips can be either in a list or Gantt chart.
To print information:
1.
Select the properties to be printed, select Continue. A printer-friendly version of the information is generated.
2.
Print the information. A printer-friendly version of the information shown on the screen is sent to a printer.
Note
The Print dialogue may vary depending on the web browser used.
